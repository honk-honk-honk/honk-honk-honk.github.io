Ok. That’s interesting. I totally forgot about this blog. Today I suddenly found it. And I found my note about autoposting. I did a little investigation about how I made this autoposting mechanism. Well, i’m thought I was smarter :) It’s pretty easy, but there is one critical dependency - it based on MacOS automator. There is requirements.

  • Static site generator. I used «Hugo».
  • Any text editor at you like.
  • MacOS automator
  • Git
  • GitHub pages hosting

Usage is very simple. Just write you post in any text editor, highlight it, and click, in the «services» sub-menu at «post to blog» button, and that’s all! Just like a «share» button, but for GitHub pages. One serious restriction - no images.

Automator process have 3 parts:

  • Create text file contains highlighted text for future usage.
  • Take post name
  • Run bash script:
cd /Users/honk/blog
/usr/local/bin/hugo new posts/$1.md
cat content/posts/$1.md ./temp.txt > ./temp.md
mv ./temp.md ./content/posts/$1.md
/usr/local/bin/hugo -D
cp -r ./public/ /Users/bca19/honk-honk-honk.github.io
cd /Users/honk/honk-honk-honk.github.io
git add -A
git commit -m "new post $1"
git push -u origin master

That’s all, folks!
